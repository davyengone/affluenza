Affluenza
-Affluenza is a practice in data visualization that maps gentrification's in NYC's 5 boroughs
-all data comes from the Census, city-data and zillow
-Sign in via Twitter

Connects to the following APIs
-Google Maps V3
-CartoDB
-Zillow
-Instagram
-Twitter
